# House_Price_Prediction

### 1. Linear Regression
### Linear Regression is a machine learning model that predicts a continuous value (such as house price) based on a set of independent variables (such as number of bedrooms, square footage, etc.). The model is a line that best fits the data points.

### 2. Gradient descent
### Gradient Descent is an iterative optimization algorithm that is used to find the minimum of a function. In the context of linear regression, gradient descent is used to find the weights and biases of the model that minimize the cost function.

### 3. Cost Function
### The cost function measures the error between the predicted values and the actual values. The goal of gradient descent is to minimize the cost function by adjusting the weights and biases of the model.

### Here are the steps involved in linear regression with gradient descent:

### i)- Choose a learning rate. The learning rate controls how much the weights and biases are adjusted in each iteration of gradient descent.

### ii)- Initialize the weights and biases. The weights and biases are randomly initialized.

### iii)- Calculate the predictions. The predictions are made by multiplying the independent variables by the weights and adding the biases.

### iv)- Calculate the error. The error is calculated by comparing the predictions to the actual values.

### v)- Calculate the gradients. The gradients are calculated by taking the derivative of the cost function with respect to the weights and biases.

### vi)- Update the weights and biases. The weights and biases are updated by subtracting the gradients from the current values multiplied by the learning rate.

### vii)- Repeat steps 3-6 until the cost function converges.
